1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

На GitHub.com перейдите на главную страницу репозитория.

Над списком файлов щелкните  Код.

щелкните SSH и скоприуйте ссылку

Откройте GIT Bash.

Измените текущий рабочий каталог на расположение, где должен находиться клонированный каталог.

Введите git clone и вставьте URL-адрес, скопированный ранее.

$ git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY

2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

заходим в нужный каталог и выполняем команды

git init "создать пустой репо"

git add . “поставить все файлы проекта на контроль”

git commit -m 'new' сохранить изменения (выполнить коммит)

git remote add origin < SSH ссылка>

git push -u origin main выгрузить на GitHub

3. Как получить на локальный компьютер изменения с github.com

При помощи команды git pull

4.Какую информацию нам дают команда :
git status отображает состояние рабочего каталога и раздела проиндексированных файлов.

git log  используется для просмотра истории коммитов, начиная с самого свежего и уходя к истокам проекта.

git remote -v   просмотр текущей привязки
